# Object Inspector
In-game object inspector for Godot 3.2.

![](https://i.imgur.com/3GEZhLV.png)

# Support types:
- Bool
- Int
- Float
- String
- Color
- Vector2
- Vector3
- Rect2
- Enum
- Flags

# Installation:
1. Clone or download this project to `addons/object-inspector` folder.
2. Enabled `Object Inspector` in Plugins.
3. Add `ObjectInspector` node to the scene.
4. Profit.
